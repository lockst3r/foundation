## Foundation Website

### QUICK START

In the project directory, you can run:

npm i

npm start

Runs the app in the development mode.
Open https://lockst3r.github.io/foundation/ to view in the browser.

The page will reload if you make edits.

The tech stack is:

1) HTML5 
2) CSS3
3) FlexBox
4) Sass(Scss)
5) BEM methodology
